# Mobile-First Rancher Wrapper (Saddle)

## Concept
The existing Rancher UI is cumbersome on mobile devices. "Saddle" acts as a thin proxy for critical management tasks (restarts, logs, health).

## Strategy
- Logic-first: Document reasoning in code/roadmap.
- Shared Library: Common DTOs for API and WASM stability.
- Mobile PWA: Future goal for true app-like experience.
